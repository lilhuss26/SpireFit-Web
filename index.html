Submit Data to Google Sheets

Setup Instructions:

  1. Create a new Google Sheet
  2. Go to Extensions > Apps Script
  3. Replace the default code with the script provided below
  4. Save the project and deploy it as a web app
  5. Set permissions to "Anyone" and copy the generated URL
  6. Replace the scriptURL in the JavaScript code below with your URL

Google Apps Script Code:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([data.name, data.number, new Date()]);
  return ContentService.createTextOutput("Success");
}
